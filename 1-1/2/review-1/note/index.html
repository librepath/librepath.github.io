学员手册（绿宝书）

认识三级修学：

  1. 缘起：根据社会和教界的需要，依五大要素构建的大众化、标准化、模式化的修学体系。
  2. 课程设置：同喜（探索生命真谛）->同修（修菩提心）->同德（禅修证悟空性）
  3. 共处规则：六和精神（戒、见、利、身、口、意和谐）。
  4. 学员公约：依据两套模式，维护修学关系清净，是大家共同修学的保障。
  5. 学员须知：进入寺院或接触其他法门时的注意事项。
  6. 修学方式：自修共修相互结合的方式，相互促进。
  7. 修学态度：真诚、认真、老实。
  8. 修学团队：性质、学训、准则、学员之歌（有法有爱）。

如何修学：

  1. 修学方式：自修、共修相结合，自修是基础，共修是提升。
  2. 修学态度：真诚、认真、老实。
  3. 修学方法：八步骤三种禅修、正念禅修
  4. 修学效果：落实正确的态度和方法之后，水到渠成地成就佛菩萨的生命品质。
  5. 自修准则：自修时应遵守的注意事项。真诚认真老实的闻思法义。
  6. 交流准则：共修时应遵守的注意事项。要检讨自己不足，随喜他人功德，及如何分享等。
  7. 班级交流流程。

信仰篇

当代宗教信仰问题的思考

  1. 宗教与人生：信仰解决人生大事，造就人格。实践信仰越早越好。
  2. 正信、迷信与邪信：明辨是非，通过闻思经教，如理思维，开启智慧。
  3. 信仰与理性、科学：佛教和科学都是注重理性，但是关注点不同，一是物质、一是精神。
  4. 宗教与社会进步：文化和经济平衡发展，道德与法制不可替代。
  5. 信仰现象的分析：宗教信仰的范畴；信仰是改善人生追求真理的动力。

佛教与中国传统文化

  1. 佛教与哲学：儒释道三家、各个时代的哲学都深受佛教影响
  2. 佛教与文学：经典本身、时空观、佛教思想内涵
  3. 佛教与艺术：雕塑、建筑、绘画、书法
  4. 佛教与民俗、道德：节日、道德观念。
  5. 人生的幸福：幸福关键在于心态
  6. “空”的智慧：“空”不是什么都没有，是缘起性空。修行上说“空”主要是为了破除我执。

人生五大问题

  1. 何为幸福：人生本质是苦，幸福是对苦的环节，佛菩萨的得大自在才是永恒的幸福。
  2. 我是谁：哲学意义上的永恒的我，是找不到的。佛教提出“无我”是透彻的人生真相。
  3. 生从何来，死往何去：提出轮回的观点，死后去向轮回，也可以通过修行跳出轮回。
  4. 何为命运：命运是自己行为决定的，行为成为习惯，习惯成为性格，进而成为人格。
  5. 活着为什么：不要再稀里糊涂混日子了，活着要有意义。

信仰与人生

  1. 信仰可以建立道德准则：佛教通过因果观和人性论，建立自他利益统一的道德。
  2. 信仰可以引导精神追求：参考佛教与中国传统文化
  3. 信仰可以使人心态超然：以缘起法审时度势、从缘起法认识空性。
  4. 信仰可以培养慈悲大爱：有我之爱是有限的、无我才能慈悲大爱
  5. 信仰可以解决生死归宿：死亡并不是结束、临终关怀
  6. 信仰可以找回自己
